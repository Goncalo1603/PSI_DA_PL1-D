Sistema de Gestão de uma Cantina

Este projeto é uma aplicação C# Windows Forms.NET para gerir uma cantina e os seus clientes, funcionários, extras, pratos, menus, reservas e muito mais.

Instalação

Para instalar a aplicação, siga estes passos:

    Clone o repositório do GitHub para a sua máquina local.
    Abra o ficheiro com a solução (Projeto.sln) no Visual Studio.

Configuração

O projeto não requer nenhuma configuração específica.

Execução

Para executar a aplicação, siga estes passos:

    Defina o projeto como projeto de inicialização no menu ao abrir o Visual Studio.
    Prima F5 para executar a aplicação.

Uso

A aplicação dispõe de vários formulários para gerir diferentes entidades:

	Form1: O formulário principal para navegar para outros formulários.
    	funcionarios: Para gerir funcionários.
    	clientes: Para gerir clientes.
    	extras: Para gerir extras.
    	pratos: Para gerir pratos.
    	multas: Para gerir multas.
    	reservas: Para gerir reservas.
